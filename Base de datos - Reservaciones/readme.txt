El objetivo de crear una base de datos es poder almacenar información relacionada a las reservaciones.
Dicha información contenida puede ser usada con fines estadísticos en pro de análisis y mejora del negocio.

La base de datos se realizó mediante el gestor pgAdmin utilizando como servicio PostgreSQL.
